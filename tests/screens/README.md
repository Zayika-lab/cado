# Скріншоти Playwright-тестів

Кожен e2e-тест зберігає PNG-знімки сюди для візуальної перевірки.
Префікс `NN_` = хронологічний порядок появи фічі, тож **нові внизу**.
Додавай нові скріни з префіксом, більшим за поточний максимум.

| # | Файл | Що показує |
|---|------|-----------|
| 01 | `01_overlap-panel-closed.png` | `fpToggle` і `loadBtn` без накладання (панель закрита) |
| 02 | `02_overlap-panel-open.png` | Те саме при відкритій файловій панелі |
